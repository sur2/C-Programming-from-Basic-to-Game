### 난수 구현

#### srand() : 전달인자를 기반으로 난수를 초기화하는 함수

#### rand() : srand로 선언된 난수를 반환하는 함수(10자리)

#### srand의 전달인자로 time 함수를 사용하여 매 시간마다 새로 생성된 난수를 rand함수로 호출한다. 

n과 a를 통해 난수의 범위 설정

```
srand(time(nuLL));
int number = rand() % n + a;
```
